- **Cross-browser extension suite for privacy and media automation** - Four Chrome/Firefox extensions — discord-purge and uninsta for bulk deletion of DMs across Discord and Instagram, filelist-ext for torrent-tracking notifications on filelist.io, and series-auto-skip for automatic intro/credits skipping on Plex and Netflix — all built from TypeScript with esbuild targeting Chrome MV3, Firefox WebExtensions, and Tampermonkey.

- **Full-stack Next.js SaaS apps on Cloudflare Workers** - Two production Next.js 16 applications deployed via @opennextjs/cloudflare to Cloudflare Workers with D1 and Tailwind v4 — a multi-tenant fleet management SaaS with Stripe billing, driver management, and maintenance deadline notifications for the Romanian market; and a corporate IT services site with six-locale i18n, admin panel, and daily Claude API-generated blog content.
- **Self-hosted k3s homelab cluster and deployed services** - GitOps-free k3s cluster with Helmfile-managed cert-manager, Prometheus/Loki/Grafana observability, and ARC runner pools for GitHub Actions CI; services running on it include CI-deployed Nextcloud (nextcloud) over a Cloudflare WARP mesh, and a GPU-accelerated Ollama stack serving Qwen 35B MoE via NVIDIA RTX 3070 passthrough on a WSL2 node.
- **Centralized reusable GitHub Actions workflow library** - shared-workflows is a public library of 12 reusable GitHub Actions workflows — covering Node.js and Python CI, Claude Code review, Chrome/Firefox extension publishing, and dependency auditing — backed by a custom ARC runner Docker image and self-hosted runner pools on the homelab k3s cluster.
- **Personal automation, monitoring, and knowledge management** - device-activity-telegram-bot sends real-time Telegram notifications on device login and unlock events with remote shutdown support; a social journaling app aggregates GitHub, Obsidian, and claude.ai activity into a weekly SQLite database and generates LinkedIn drafts via Claude CLI; and a PARA-structured Obsidian vault with Shell and PowerShell scripts for sync and graph visualization.
